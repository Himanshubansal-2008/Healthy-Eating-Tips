Project is created using html and css . It aims to promote awareness about healthy eating habits and maintaining a balanced lifestyle. The website includes different sections such as an introduction, about section, healthy eating tips, a food gallery, and a contact form. It provides information on how to eat mindfully, stay hydrated, balance meals, and build better food habits. The design is clean, modern, and mobile-friendly, with smooth scrolling and image-based layouts to make it visually appealing. This project demonstrates the use of HTML structure, CSS grid and flexbox for layout design, and responsive web development techniques.
